# 🏏 IPL Final Score Predictor

An AI-powered IPL score prediction system built using Machine Learning and Streamlit.

---

## 📌 Project Overview

This project predicts the final score of an IPL innings based on:

* Current Score
* Overs Completed
* Wickets Lost
* Batting Team
* Bowling Team
* Current Run Rate

The system uses an XGBoost Regressor trained on IPL historical match data.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/chaithanyachary2003-cyber/ipl-score-predictor.git
```

Move into the project folder:

```bash
cd ipl-score-predictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
